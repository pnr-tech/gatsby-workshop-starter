# Gatsby Starter - DSC KITRC WORKSHOP

## Installation

Fork repository

Clone repository:

    git clone your-github-repo-url

Install Dependencies:

    npm install

To Develop this site run:

    gatsby develop

To Build after development is complete:

    gatsby build
